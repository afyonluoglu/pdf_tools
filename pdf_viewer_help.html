Professional PDF Viewer

Python ile geliştirilmiş, modern karanlık tema arayüzlü, kapsamlı PDF görüntüleyici ve metin işleme uygulaması.

🐍 Python 3.x ✅ CustomTkinter 📦 PyMuPDF 🔊 Edge TTS

🏠 Genel Bakış

Professional PDF Viewer, PyMuPDF motoru ile yüksek kaliteli PDF görüntüleme, çok türlü annotation desteği, gelişmiş metin çıkarma ve Türkçe metin-seslendirme (TTS) özelliklerini tek bir arayüzde sunar.

📄

Yüksek Kalite Render

PyMuPDF ile piksel-hassas sayfa görüntüleme; %10–%500 zoom, sayfa döndürme.

🖍️

Annotation Araçları

Vurgulama, not, işaret ve link annotation'ları; renk seçici ve annotation yönetimi.

🔍

Gelişmiş Arama

Tüm sayfalarda metin tarama; kırmızı çerçeveli vurgulama, "Next" ile sıralı gezinme.

📤

Metin Çıkarma

PDF sayfalarından Türkçe-uyumlu metin çıkarma; akıllı satır birleştirme.

✏️

Profesyonel Editör

Bul-değiştir, undo/redo, satır numaraları, font boyutu, istatistik çubuğu.

🔊

Türkçe TTS

Microsoft Edge TTS; Emel / Ahmet sesleri, 3 hız, MP3 dışa aktarım.

Toolbar'daki ↻ butonuna her tıklamada sayfa 90° döndürülür (0° → 90° → 180° → 270° → 0°).

🔍 Metin Arama

  1. Toolbar'daki arama kutusuna aranacak metni yazın (veya Ctrl+F ile odaklanın).

  2. Enter tuşuna basın veya 🔍 butonuna tıklayın.

  3. Tüm sayfalarda tarama yapılır; bulunan eşleşmeler kırmızı çerçeve ile işaretlenir ve ilk eşleşmenin bulunduğu sayfaya otomatik gidilir.

  4. "Next" butonuna basarak sonraki eşleşmeye sıralı geçiş yapın.

ℹ️

Arama büyük/küçük harf duyarlıdır. Tüm eşleşmeler terminale de raporlanır (🚩 Sonuç N: Sayfa X, Konum: ...).

🖍️ Annotation Araçları

🖍️ Metin Vurgulama

  1. İsteğe bağlı olarak 🎨 Renk butonuyla vurgulama rengi belirleyin.

  2. 🖍️ İşaretle butonuna tıklayarak vurgulama modunu etkinleştirin (durum çubuğu "Vurgulama modu açık" yazan görünür).

  3. PDF canvas üzerinde fare ile dikdörtgen çizerek vurgulamak istediğiniz alanı seçin.

  4. Modu kapatmak için tekrar 🖍️ İşaretle butonuna basın.

📤 PDF'den Metin Çıkarma

  1. PDF dosyasını açın ve toolbar'daki 📄 Metne Çevir butonuna tıklayın.

  2. Sayfa aralığı seçme penceresi açılır — Tüm sayfaları çevir veya Sayfa aralığı belirt (örn. 3–15) seçeneğini işaretleyin.

  3. Metne Çevir butonuna tıklayın. Progress bar ile ilerlemeyi takip edin.

  4. İşlem tamamlandığında metin otomatik olarak Profesyonel Metin Editörü'nde açılır.

🧹 Akıllı Metin Temizleme

Çıkarma işlemi sırasında uygulanan otomatik temizlik adımları:

  • Satır içi gereksiz kırılmalar birleştirilir (paragraf yapısı korunur)
  • Hece bölünmeleri (kelime-\n devam) akıllıca birleştirilir
  • Ardışık boş satırlar tek satıra indirilir
  • Her sayfa için === SAYFA N === ayraçları eklenir
⚠️

Taranmış (görüntü tabanlı) PDF'lerden metin çıkarma desteklenmez. Böyle dosyalar için OCR aracı gereklidir.

🔊 Metin Seslendirme (TTS)

Metin Editörü, Microsoft Edge TTS teknolojisini kullanarak Türkçe seslendirme yapar. İnternet bağlantısı gerektirir.

⚡ Hız Seçenekleri

HızRate Değeri

  1. Editörde seslendirmek istediğiniz metni seçin (seçim yoksa tüm metin kullanılır).

  2. Toolbar 2'den ses (Emel/Ahmet) ve hız (Yavaş/Normal/Hızlı) seçin.

  3. 🔊 Seslendir butonuna tıklayın. TTS durum etiketi "Ses oluşturuluyor..." mesajını gösterir.

  4. Seslendirme arka planda çalışır; arayüz bloklanmaz.

  5. Durdurmak için ⏹️ Durdur butonuna tıklayın.

  6. MP3 dosyası oluşturmak için 💾 MP3 Kaydet butonuna tıklayın ve kayıt konumunu seçin.

ℹ️

TTS özelliği için edge-tts ve pygame kütüphanelerinin kurulu olması gerekir: pip install edge-tts pygame

💾 Dosya İşlemleri

📂 PDF Açma

  1. 📁 Aç butonuna tıklayın veya Ctrl+O tuşlayın.

  2. Dosya seçici diyaloğundan .pdf dosyasını seçin.

  3. Progress bar penceresi açılır; büyük dosyalar thumbnail üretimi dahil kademeli yüklenir.

  4. Yükleme tamamlanınca alt durum çubuğu dosya adı ve sayfa sayısını gösterir.

💾 PDF Kaydetme

  1. 💾 Kaydet butonuna tıklayın veya Ctrl+S tuşlayın.

  2. Dosya adı ve konum seçin.

  3. Tüm annotation'lar PDF'e gömülü olarak kaydedilir (garbage=4, deflate=True ile optimize edilmiş).

🚀 Kurulum

Gereksinimler

pip install customtkinter>=5.2.0 PyMuPDF>=1.23.0 Pillow>=10.0.0

TTS özelliği için (isteğe bağlı):

pip install edge-tts pygame

Veya requirements.txt ile tümünü:

pip install -r requirements.txt

Çalıştırma

python pdf_view_main.py

Windows'ta start_pdf_viewer.bat dosyasına çift tıklayarak da başlatılabilir.

🛠️ Hata Yönetimi

Uygulama global bir exception handler kullanır. Tüm beklenmedik hatalar:

  • Terminal/konsola detaylı stack trace ile yazdırılır
  • pdf_viewer_errors.log dosyasına tarih-damgalı olarak kaydedilir
Hata TürüDavranış
PDF yükleme hatasıProgress penceresi kapatılır; kullanıcıya mesaj gösterilir
PDF kaydetme hatasıHata mesaj kutusu gösterilir
Annotation ekleme hatasıTerminal + mesaj kutusu ile bildirilir; kaydetme devam eder
TTS hatasıDurum etiketi güncellenir; seslendirme iptal edilir
Genel exceptionGlobal handler yakalar, log'a yazar, uygulama çalışmaya devam eder
💡

Herhangi bir hata durumunda önce pdf_viewer_errors.log dosyasını inceleyiniz. Her çalıştırmada log dosyası sıfırdan başlatılır.
